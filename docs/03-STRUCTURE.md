# Ֆայլերի կառուցվածք (փաստացի repo root)

```
maylercrm/
├── app/                 # Next.js App Router (էջեր, API routes)
├── components/          # React կոմպոնենտներ
├── lib/                 # Օգտակարություններ, Prisma, auth, R2
├── prisma/              # schema.prisma, migrations
├── scripts/             # tsx սկրիպտներ (seed, import, export)
├── public/              # Ստատիկ ակտիվներ
├── docs/                # Նախագծի փաստաթղթեր (այս թղթապանակը)
├── reference/           # Կաղապարներ, հարթակներ, checklist-ներ
└── Documents/           # Աշխատանքային ինդեքս, պատմություն, հին սպեցիֆիկացիաներ
```
